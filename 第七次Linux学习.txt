## 一、文件查找
```bash
# 进入练习目录
cd ~/Linux-study

# 1. find：精准查找文件/文件夹
find . -name "*.txt"        # 当前目录查找所有 .txt 文件
find . -name "study.txt"    # 精确查找 study.txt
find ~ -name "test*"        # 主目录查找所有 test 开头的文件

# 2. which：查找系统命令的位置
which ls
which git
which python3
which nano

# 3. locate：快速查找
sudo updatedb               # 第一次用必须执行
locate study.txt
```
