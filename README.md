# Linux 系统管理学习笔记

## 📌 项目简介
本仓库记录 Linux 系统管理与运维的学习过程，涵盖常用命令、文件权限、用户管理、进程监控、网络配置、压缩打包、链接、Shell 脚本基础等模块。

## 🛠 环境
- 操作系统：Ubuntu（虚拟机）
- Shell：Bash

## 📝 内容索引

| 次数 | 学习主题 | 主要命令/知识点 |
|------|----------|----------------|
| 第 1 次 | 路径导航 | `pwd`, `cd`, `ls` |
| 第 2 次 | 文件与目录操作 | `mkdir`, `touch`, `cp`, `mv`, `rm`, `rm -r` |
| 第 3 次 | 文件查看与编辑 | `cat`, `head`, `tail`, `less`, `nano` |
| 第 4 次 | 权限管理 | `ls -l`, `chmod`, `sudo`, `apt` |
| 第 5 次 | 网络与系统监控 | `ping`, `ip addr`, `ps aux`, `top`, `df -h` |
| 第 6 次 | Git 基础 | `git add`, `git commit`, `git push` |
| 第 7 次 | 文件查找 | `find`, `which`, `locate`, `sudo updatedb` |
| 第 8 次 | Vim 编辑器 | `vim`, `i` (插入), `Esc` (退出), `:wq`, `:q!` |
| 第 9 次 | 文本处理 | `grep`, `wc`, `sort`, `uniq` |
| 第 10 次 | 压缩打包 | `tar -zcvf`, `tar -zxvf`, `zip`, `unzip` |
| 第 11 次 | 软链接 | `ln -s`, `rm` |
| 第 12 次 | 文件归属 | `chown` |
| 第 13 次 | 重定向与管道 | `>`, `>>`, `\|` |
| 第 14 次 | 系统信息查看 | `uname -a`, `cat /etc/os-release`, `free -h`, `whoami`, `hostname` |
| 第 15 次 | 磁盘空间查看 | `df -h`, `du -h` |
