# Linux 系统管理学习笔记

## 项目简介
本仓库记录 Linux 系统管理与运维的学习过程，涵盖常用命令、权限管理、进程监控、网络配置、Shell 脚本等模块。

## 环境
- 操作系统：Ubuntu / CentOS
- Shell：Bash

## 内容索引
- **文件与权限管理**：ls、chmod、chown、umask
- **用户与组管理**：useradd、usermod、passwd、groupadd
- **进程监控与管理**：ps、top、kill、systemctl
- **网络配置与排查**：ip、netstat、ping、traceroute
- **软件包管理**：apt、yum、dpkg
- **Shell 脚本基础**：变量、条件判断、循环、crontab 定时任务

## 关联项目
本仓库为简历中"Linux 系统管理与运维辅助实践"项目的支撑仓库之一，展示 Linux 系统管理基础能力。
